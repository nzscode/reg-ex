RegEx Basics

Flags

  • No Flags
    • Returns the first pattern match, case sensitive
      • Expression:

        /cat/

  • Global
    • Allows matching anywhere in the string
      • Expression:

        /cat/g

  • Case Insensitive
    • Allows matching anywhere in the string, without matching case
      • Expression:

        /cat/i

  • Multiline
    • Allows matching in multiple strings
      • Expression:

        /cat/m

Below is a table of the most commonly used RegEx tokens

Anchors

Tokens Definition
^ Start of string , or start of line in multiline pattern
$ End of string, or end of line in multiline pattern

Character Classes

Tokens Definition
\s Whitespace
\S Not whitespace
\d Digit
\D Not digit
\w Word
\W Not word

Quantifiers

Tokens Definition Example Explanation
* 0 or more {3} Exactly 3
+ 1 or more {3,} 3 or more
? 0 or 1 {3,5} 3, 4 or 5
Add a ? to a quantifier to make ir ungreedy(means if it exists then the pattern applies)

Escape Sequences

Tokens Definition
\ Escape following character
Excaping allows the special character to be matched.
Common Metacharacters
^ [ . $
{ * ( \
+ ) | ?
< >
The escape character is usually \

Pattern Modifiers (Flags)

Tokens Definitions
g Global match (Case sensisitive)
i Case - Insensitive
m Multiline

Groups & Ranges

Tokens Definitions
. Any character except new line (\n)
(a|b) a or b
(...) Group
(?:...) Non-Capturing (Passive) group
[abc] Range (a or b or c)
[^abc] Not (a or b or c)
[a-q] Lower case letter from a to q
[A-Q] Uppercase letter from A to Q
[0-7] Digit from 0 to 7
Ranges are inclusive

Special Characters

Tokens Definitions
\n New Line
\r Carriage return
\t Tab

Assertions

Tokens Definitions
?= Lookahead assertion
?! Negative Lookahead
?>= Lookbehind assertion
?!= OR ?<! Negative lookbehind
?() Condition [if then]
?()| Condition [if then else]
?# Comment

String Replacement

Tokens Definitions
$n nth non-passive group
$2 "xyz" in/^(abc(xyz))$/
$1 "xyz" in /^(?:abc)(xyz)$/
(The first group: (?:abc) is a non-passive group)
$` Before matched string
$' After matched string
$+ Last matched string
$& Entire matched string
Some RegEx implementations use \ instead of $
